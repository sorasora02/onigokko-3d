# onigokko-3d

## 概要

Unityで制作した3D鬼ごっこゲームです。
プレイヤーはTPS視点でステージ内を移動し、追いかけてくる敵から逃げながらコインを集めます。
制限時間まで逃げ切ることを目標にした、シンプルな3Dアクションゲームです。

## ゲームルール

- 敵に捕まるとゲームオーバー
- 制限時間まで逃げ切るとクリア
- ステージ内のコインを取るとスコアが増える
- ゲーム終了後に最終スコアを確認できる

## 操作方法

| 操作 | 内容 |
| --- | --- |
| W / A / S / D | 移動 |
| マウス移動 | 視点操作 |
| R | リスタート |
| Escape | 終了確認 |

## 使用技術

- Unity
- C#
- Unity Input System
- TextMeshPro

## 実装した主な機能

- TPS視点のカメラ
- カメラの向きを基準にしたプレイヤー移動
- 敵の追跡
- コイン取得によるスコア加算
- 制限時間によるクリア判定
- 敵との接触によるゲームオーバー判定
- リスタート機能
- 終了確認機能
- 操作説明UI
- 最終スコア表示

## 工夫した点

### カメラ基準の移動

Wキーを押したときにワールド座標の前方向へ進むのではなく、カメラから見た前方向へ進むようにしました。
これにより、TPSゲームとして直感的に操作できるようにしています。

### スコア要素

ただ逃げるだけではなく、コインを集めることでスコアが増えるようにしました。
敵から逃げるリスクと、コインを取りに行くリターンを作ることを意識しています。

### 3Dステージ

平面だけでなく、壁や高低差のあるステージにすることで、3D空間を使った逃走ルートを作れるようにしました。

## 制作目的

3Dゲームまたは3Dグラフィックスを用いたゲームとして提出するために制作しました。
短期間で完成させることを優先し、基本的なゲームループである「開始、操作、逃走、スコア取得、終了判定」が成立する形を目指しました。
